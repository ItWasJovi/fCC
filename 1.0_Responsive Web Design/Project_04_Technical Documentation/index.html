Flexbox

One of the easiest ways to create a multi-column layout is using Flexbox

.flex-grid { display: flex; } .col { flex: 1; }

Depending on how many div columns you apply this class to, you'll have equal-sized columns.

As a general rule it's good to not have more than 12 columns in total.

Alignment

For Flexbox centering, you just need to use the two centering properties:

.parent { display: flex; justify-content: center; align-items: center; }

For block-level elements, just use auto margins:

.center { margin: 0 auto; }

Of course there are other elements that you might want to center such as:

  • Inline elements: Text, links
  • Multiple block-level elements
  • Vertical centering

A quick tip for vertically centering text is to make the line height equal to the container height:

.container { height: 100px; line-height: 100px; }
