A brief summary of Kygo's career:

  • 2014 - Kygo's single "Firestone" is released, featuring vocals by Conrad Sewell
  • 2015 - Cloud Nine, Kygo's debut album, is released to critical acclaim
  • 2017 - An EP titled "Stargazing" is released featuring collaborations with Selina Gomez, Ellie Goulding, and Justin Jesson
  • 2017 - Kygo releases his second album, "Kids in Love"
  • 2018 - With rumours of a new album, two new singles are released - "Born to Be Yours" and "Happy Now"
